

进入主 文件夹
    1) 安装所依赖的npm包：
        在当前目录中打开命令行窗口 输入 " npm install "
    2).连接数据库:
         在当前目录中打开命令行窗口，输入 " mongod --dbpath ./data " 连接数据库
    3）运行服务器程序
        在当前目录中 再打开另外一个命令行窗口， 输入 " npm start "
    4) 打开网页：
        打开chrome浏览器，输入网址 " localhost:3000 ", 会自动跳转到 登陆页面

功能实现：
  1）用户：用户的注册、登录、登出 （在导航栏右方）
  2) blog的评论:
      采用 @某人 的方法进行回复
